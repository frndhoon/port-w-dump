## port-w-dump

> 덤프 페이지 작업

### 폴더 구조

```
src/
├── app/           # Next.js App Router - 페이지 라우팅
├── component/     # UI 컴포넌트 - 재사용 가능한 화면 구성요소
│   └── shadcn-ui/ # shadcn/ui 컴포넌트들
├── asset/         # 정적 리소스 - 폰트, 이미지 등
│   └── font/      # 폰트 파일들
└── lib/           # 라이브러리 설정 - 외부 라이브러리 설정
```

- 분류 기준: 코드의 역할/타입별로 구분
